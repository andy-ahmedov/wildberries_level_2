Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main
import (
	"fmt"
)
func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}
func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод: [3 2 3]

Слайс состоит из указателя на базовый массив, поля длины и ёмкости.
В программе мы передали в modifySlice не указатель на структуру слайса, а саму структуру, следовательно структура скопировалась вместе с указателем на базовый массив, поэтому операции изменения массива будут актуальны за пределами области функции modifySlice до того, как базовый массив измениться после расширения слайса за его пределы, после уже они актульны не будут, поэтому слайс стоит передавать по ссылке.
```
